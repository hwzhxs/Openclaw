## CRITICAL Implementation Details

### Lenis + GSAP Integration (scroll.ts)
```typescript
// The key integration pattern - must do this:
lenis.on('scroll', ScrollTrigger.update)
gsap.ticker.add((time) => { lenis.raf(time * 1000) })
gsap.ticker.lagSmoothing(0)
ScrollTrigger.scrollerProxy(document.documentElement, {
  scrollTop(value) { return value !== undefined ? lenis.scrollTo(value) : lenis.scroll },
  getBoundingClientRect() { return { top: 0, left: 0, width: window.innerWidth, height: window.innerHeight } }
})
```

### Three.js Neural Mesh (NeuralMesh.tsx)
- 4 sphere nodes at tetrahedral positions, each agent's emissive color
- ~100 particles orbiting between nodes (Points geometry)
- Connecting lines between nodes (LineBasicMaterial)
- Subtle bloom via EffectComposer (UnrealBloomPass strength=0.3)
- Continuous gentle rotation
- On scroll: camera pushes forward slightly
- Dynamic import with ssr:false in Hero.tsx
- Pauses rendering when not in viewport
- Mobile: component not loaded at all (use HeroFallback instead)

### Scroll Pins
Use GSAP ScrollTrigger with `pin: true, scrub: true`:
- Hero: pinned 1.5x viewport scroll
- Squad: pinned 4x viewport scroll (each agent = 25% of pin duration)
- Terminal: pinned 2x viewport scroll
