# Design Taste Reference

## Gold Standard References
Products to benchmark against when reviewing design work:

| Category | Reference | Why |
|----------|-----------|-----|
| Clean UI | Linear, Notion | Minimal, purposeful, every pixel earns its place |
| Typography | Stripe docs, Apple | Perfect hierarchy, readable, elegant |
| Data display | Vercel dashboard, Grafana | Dense info, still scannable |
| Marketing | Apple.com, Stripe.com | Storytelling through layout and whitespace |
| CLI/Terminal | Charm.sh tools | Even terminals can be beautiful |
| Docs | Tailwind docs, Stripe API | Searchable, scannable, great examples |

## Taste Calibration Questions
Ask these during every design review:

1. **Would I screenshot this to share as inspiration?** If no → not beautiful yet
2. **Does it make me FEEL something?** No feeling = no soul. Push for emotion.
3. **Does it have ONE strong opinion?** Great design makes a choice and commits. Mediocre design tries to please everyone
4. **Where could this be BOLDER?** Always ask — the first version is almost never bold enough
5. **Is there a moment of delight or surprise?** The best designs have one unexpected detail that sticks with you

## Design Review Verdict Scale

- 🏆 **Exceptional** — Screenshot-worthy. Ships as portfolio piece.
- ✅ **Solid** — Professional, clean, no complaints. Ships confidently.
- ⚠️ **Functional** — Works but doesn't impress. Needs polish pass.
- 🔄 **Needs Work** — Specific issues identified. Revision required.
- ❌ **Start Over** — Fundamental problems (wrong direction, not just surface issues).

_(patterns, insights, and disagreements that sharpened our taste)_

## Anti-Patterns to Watch For

### "The Kitchen Sink"
Everything the designer knew, thrown on one page. Fix: ruthlessly cut.

### "The Clone"
Copied a template but didn't adapt it. Feels generic. Fix: find the ONE thing that makes this unique and amplify it.

### "The Overcompensation"
Too many animations, gradients, shadows to hide weak layout. Fix: strip to skeleton, fix structure first, then add tasteful polish.

### "The Afterthought"
Functionality is great, design was last-minute. Usually exposed by: inconsistent spacing, mixed icon styles, no loading/empty states. Fix: design system pass.
