# Shared Taste Library

Location: `C:\Users\azureuser\shared\taste\`

This is the team's shared design taste — so all agents speak the same visual language.

## Files
- `DESIGN-SYSTEM.md` — Core design system (Apple HIG-inspired): typography, color, spacing, animation, components, anti-patterns
- `references/` — Screenshots and links of sites we admire (add over time)
- `patterns/` — Reusable design patterns and code snippets (add over time)

## Who Uses What
| Agent | How to use |
|---|---|
| **Admin** 👮 | Reference when writing task briefs — include design direction |
| **Thinker** 🧠 | Reference when writing specs — ensure proposals match our taste |
| **Creator** 🛠️ | Follow strictly when building — this is your design bible |
| **Gatekeeper** 🛡️ | Reference when reviewing — reject anything that doesn't meet the bar |

## Rules
- All agents read `DESIGN-SYSTEM.md` when working on anything visual
- Only update via team consensus (propose changes → Gatekeeper approves)
- Add reference sites and patterns as we discover them
