# shared/context

Team-shared context for OpenClaw agents: rules, runbooks, templates, and knowledge.

## Start here
- `INDEX.md` (entry + routing rules)
- `TEAM-MEMORY.md` (small set of critical team consensus)

## Knowledge library
- `knowledge/` (topic-based reusable notes)

## Contributing rule of thumb
- If you want future sessions/agents to know it: **write it down**.
- If it’s reusable: put it in `knowledge/`.
- If it’s a rule: also update `TEAM-MEMORY.md`.
