# XHS Content Strategy — Fashion + Tennis (Women) + Lookbook

**Owner:** Admin (xXx)
**Last updated:** 2026-03-03 (UTC)

## Positioning
- Target niche: women’s tennis fashion with **Kith×Wilson** vibe + **Aimé Leon Dore (ALD) lookbook/editorial** style.
- Thesis: “court editorial / lookbook-style tennis outfit formulas”（能打球，也能进城）

## Content shelf (栏目货架)
- 60%：网球穿搭公式（可复制公式/系列）
- 20%：高级感细节课（配色/面料/比例/配饰）
- 20%：单品/装备清单（鞋/裙/球包/防晒等）

## High-end feel system (高级感 = 一致性 + 克制 + 证据)
- Consistency:
  - Fixed palette: cream / navy / forest green / grey-blue
  - Fixed composition: full-body 3/4 + 2 detail shots
  - Fixed typography placement
- Restraint:
  - fewer logos/props/text
  - more “结论 + 理由 + 清单”
- Proof:
  - natural light
  - fabric/detail closeups
  - actionable lists

## Data-backed notes (from sampled thread results; see research.md for tables)
- Lookbook/styling赛道：收藏率常常很高，甚至出现“收藏 > 点赞”。
- 图文在 lookbook 赛道更利于沉淀粉丝；视频可作为补充。
